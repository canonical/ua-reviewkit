UA Reviewkit Documentation
==========================

The UA Reviewkit contains tools and helpers used as part of Ubuntu Pro Support
Handover reviews.

A set of tools and scripts are provided to assist the process of collecting
data, running tests and applying assertions to resources and artifacts
delivered as part of a handover review.
